# 视频客服Demo
实现了一个带排队功能的视频客服场景示例，包括呼叫、排队、音视频沟通、屏幕共享、聊天功能。

## 支持的平台
示例应用包含 Android, iOS, Windows 以及Web平台。

展示了：
- 在线状态
- 呼叫
- 客户排队
- 坐席端客户自动分配
- 坐席端客户手动分配(免打扰)
- 打开 / 关闭麦克风
- 打开 / 关闭摄像头
- 打开 / 关闭扬声器
- 设置分辨率，帧率，码率
- 房间内文字聊天
- 云端录制
- 拍照

项目文件夹名称|平台|描述
